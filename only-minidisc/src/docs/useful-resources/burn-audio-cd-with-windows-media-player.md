---
title: Burn Audio CD with Windows Media Player
---

insert blank CDR-RW
open Windows Media player
click burn tab

Do import playlist or drag music
rename
select burn options then Audio CD
Start burn
1 burn

Adapter
SONY AC-E45HG AC DC Adapter 4.5V 700mA 10W Power Supply for walk
